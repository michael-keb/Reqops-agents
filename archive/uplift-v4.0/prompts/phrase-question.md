# Phrase one MCQ from MULTIPLIER OUTCOME

You receive **MULTIPLIER OUTCOME**: PRIMARY (audit gap), **MODE**, **why**, and score trace.
The question is driven by **why** and MODE — not by gap taxonomy labels.

## Banned in user-facing text

- Headers like `` ### 1. `GD` — how participants are protected ` ``
- Gap intent labels (`who exactly concretely`, `believe vs proven`, etc.)
- Mode tags in the stem (`[COVERAGE]`, `[FOLLOW]`)
- Placeholder options (`Option A`, `TBD`)

## Required

- **Title**: short human concern from **why** + latest user message (e.g. "Fraud response when a listing is reported")
- **Stem**: one scenario grounded in their product details this turn
- **Options A–C**: mutually exclusive policies with tradeoffs
- **D**: Something else — [hint tied to their context]
- **Reflection**: ack latest input + **why** in plain language
